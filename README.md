# homepage-next

Next.js candidate for the Vishal Builds homepage revamp.

## Purpose

This folder is the new homepage implementation promoted from `vishal-lab/apps/homepage-next` for careful production testing before replacing the current static homepage.

## Stack

- Next.js
- React
- Tailwind CSS
- Framer Motion

## Local commands

```powershell
npm install
npm run dev
```

## Notes

- Intended final domain: `https://www.vishalbuilds.com`
- Keep `homepage-vercel` as the fallback/static production version until this Next.js version is verified live.
